# Shouvre - Luxury Fashion Store Landing Page

## Original Problem Statement
Build a beautiful landing page for a Shopify store similar to suzannah.com for the development store https://shouvre.myshopify.com/ with password "maubra".

## Brand Identity
- **Store Name**: Shouvre
- **Color Palette**: 
  - Primary Purple: #4C1C5B
  - Accent Gold: #E1921D
- **Style**: Luxury fashion with elegant, sophisticated aesthetic

## User Personas
1. **Fashion Conscious Shopper** - Looking for high-quality, timeless pieces
2. **Luxury Buyer** - Values premium materials and craftsmanship
3. **Newsletter Subscriber** - Interested in exclusive updates and promotions

## Core Requirements
### Frontend Only (Current Phase)
1. Responsive landing page inspired by suzannah.com
2. Hero banner with featured collection
3. Featured collections section (4 collections)
4. About/Brand story section
5. Newsletter signup
6. Header with navigation
7. Footer with links and social media

### Technical Stack
- Frontend: React with shadcn/ui components
- Styling: Tailwind CSS
- Mock Data: Local JSON data structure
